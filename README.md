## Time endless web app


### Dev

- `./gradlew build`
- `./gradlew clean bootRun`
- `java -jar build/libs/time-endless-app-0.3.0.jar`

=== Using sbt as the build tool

Start up sbt console

[source, bash]
----
sbt
----

Start App

[source, bash]
----
~re-start
----
